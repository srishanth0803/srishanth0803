## 🚀 About Me

I'm **Gottam Srishanth**, a data-driven Python developer who loves turning messy datasets into clean, actionable insights. I build with **pandas**, **NumPy**, and **MySQL**, and I visualize stories with **Matplotlib** and **Seaborn**.

Beyond code, I create content to share what I learn — you'll find me on **YouTube**, **Medium**, and **Instagram** breaking down data analysis concepts for others.

- 🔭 Currently deep-diving into data analysis & visualization projects
- 📊 Passionate about finding the story hidden inside data
- ✍️ I write and post content to teach what I learn
- ⚡ Fun fact: I'd rather debug a query at midnight than leave it unsolved
